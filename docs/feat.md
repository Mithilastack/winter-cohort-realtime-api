# Features

## Chat Management

    1. Chat should persist after page reload
    2. Chat updates should be saved in real-time
    3. User can clear entire chat manually
    4. System should handle streaming updates correctly
    5. Storage should stay in sync with UI state

## Behaviour and Controls

    1. User can choose AI tone (e.g., friendly, formal, teacher, interviewer)
    2. User can control response length (short, normal, detailed)
    3. AI behavior updates instantly without resetting the conversation
    4. UI clearly displays current behavior settings
    5. System preserves selected behavior during the session
